🛠️ Configuration Guide [+]

Ripple is configured locally via a JSON flat-file. To manage projects:

  1. Open ripple.config.json in the project root.
  2. Add Project: Append a new project object to the "projects" array with its key, paths, goals, and events.
  3. Edit Project: Modify goals or events to match your intent.
  4. Run Analyzer: Run python scripts/analyze.py in the terminal to compile metrics.

The dashboard dynamically reloads the generated analytics.
